¿Qué hace FLAI?

  • 💡 Detección: métrica bidimensional (Equidad/Igualdad) para auditar modelos.
  • 📈 Creación: genera un grafo causal interpretable del problema.
  • 🛡️ Mitigación: ajusta relaciones y tablas de probabilidad (en‑training) y genera fair data (pre‑training).

Basado en “Quantifying algorithmic discrimination: A two-dimensional approach to fairness in AI”.
Consulta la cita académica en el README y el paper original.

Por qué esta métrica

Separa Igualdad (paridad por grupos) de Equidad (ajuste a necesidades diferenciales), ofreciendo una señal más informativa que métricas tradicionales.

Creación causal

Construye grafos causales y ajusta relaciones sensibles para evitar la propagación de sesgos.

Mitigación transparente

Genera fair data y entrena modelos sobre él para reducir disparidades manteniendo rendimiento.
